# Teaching

[Autonomous Robots](https://moodle.upm.es/titulaciones/oficiales/course/view.php?id=2350). The course summarizes the latest software approaches in autonomous robotics. After a brief introduction to the classical and behavior-based architectures, we apply state-of-art hybrid techniques to complete a mini project in a simulated environment.
